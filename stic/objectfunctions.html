Object Functions

ObjectFunctions are functions that you can "attach" to parts!

Functions:

Stic.ObjectFunction.New(object : Instance, funcitonName : string, func : function)

Stic.ObjectFunction.Run(object : Instance, funcitonName : string)

Stic.ObjectFunction.Get(object : Instance, funcitonName : string)

Stic.ObjectFunction.GetAll(object : Instance)

Example usage:

Script1:


local Stic = require(game.ReplicatedStorage.Stic)
local part = workspace.Part

function toGreen()
    part.Color = Color3.new(0,1,0)
end

Stic.ObjectFunction.New(part, 'ToGreen', toGreen)

Script2:


local Stic = require(game.ReplicatedStorage.Stic)
local part = workspace.Part

Stic.ObjectFunction.Run(part, 'ToGreen')

--_________________________________--
-- other way:
local Stic = require(game.ReplicatedStorage.Stic)
local part = workspace.Part

local toGreen = Stic.ObjectFunction.Get(part, 'ToGreen')
toGreen()

--_________________________________--
-- another way:
local Stic = require(game.ReplicatedStorage.Stic)
local part = workspace.Part

local partFunctions = Stic.ObjectFunction.GetAll(part)
partFunctions['ToGreen']()
partFunctions['someOtherObjectFunctionName']()
partFunctions['someOtherObjectFunctionName']()
