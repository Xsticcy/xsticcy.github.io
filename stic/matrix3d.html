Matrix3D

Matrix3D is a way to generate three dimensional tables with one function!

Functions:

Stic.Matrix3D.NewMatrix(matrixName : string, size : Vector3)

Stic.Matrix3D.GetMatrix(matrixName : string, size : Vector3)

matrix3d.Set(position : Vector3, value : any)

matrix3d.Get(position : Vector3)

Example usage:


local Stic = require(game.ReplicatedStorage.Stic)
local matrix3d = Stic.Matrix2D.NewMatrix('MyMatrix3D', Vector3.new(30,30))
matrix3d.Set(Vector3.new(1,1,1), 'someValue')
matrix3d.Set(Vector3.new(2,1,1), 30)
print(matrix3d:Get(Vector3.new(3,1,1))) -- prints 0 because nothing is there
local data = Stic.Matrix3D.GetMatrix('MyMatrix3D')
print(data[1][1][1]) -- prints 'someValue'
print(data[2][1][1]) -- prints 30
