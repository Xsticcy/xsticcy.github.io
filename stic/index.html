This is the documentation of Stic.





What is Stic?

Stic is a Roblox module, and it makes programming games faster and easier.





Why Stic?

Stic is super easy to learn and use. It can make your game more organized and run faster!





How Stic makes my game faster and more organized?

With Stick you can get rid of a lot of loops and modulescripts.

Stick has a signal system to communicate between scripts so you don't need to make Bindable Fuctions and loops for communication.

With Stick, you can get rid of every object based value(Intvalues, BoolValues, etc), because Stic has a way, to make shared variables easily.

Stick also has a custom service system, so you can create your own services and use them in any script!
