Matrix2D

Matrix2D is a way to generate two dimensional tables with one function!

Functions:

Stic.Matrix2D.NewMatrix(matrixName : string, size : Vector2)

Stic.Matrix2D.GetMatrix(matrixName : string, size : Vector2)

matrix2d.Set(position : Vector2, value : any)

matrix2d.Get(position : Vector2)

Example usage:


local Stic = require(game.ReplicatedStorage.Stic)
local matrix2d = Stic.Matrix2D.NewMatrix('MyMatrix2D', Vector2.new(30,30))
matrix2d.Set(Vector2.new(1,1), 'someValue')
matrix2d.Set(Vector2.new(2,1), 30)
print(matrix2d:Get(Vector2.new(3,1))) -- prints 0 because nothing is there
local data = Stic.Matrix2D.GetMatrix('MyMatrix2D')
print(data[1][1]) -- prints 'someValue'
print(data[2][1]) -- prints 30
