Shared groups

Shared groups are a group of are variables you can access between any script.

Functions:

Stic.NewSharedGroup(groupName : string)

group.AddValue(valueName : string, value : any)

Example usage:

Script1:


local Stic = require(game.ReplicatedStorage.Stic)
local someGroup = Stic.NewSharedGroup('someGroup')
someGroup.AddValue('someValue', 150)

Script2:


local Stic = require(game.ReplicatedStorage.Stic)
wait() -- we wait so the group is created first
print(Stic.Shared.someGroup.someValue) -- prints 150

Shared groups don't seem like a very useful thing, but they can make your code & game more organized.
