Custom Services

Custom Services are a group of functions and variables.

Functions:

Stic.NewService(serviceName : string)

Stic.GetService(serviceName : string)

service.AddFunction(funcName : string, func : function)

service.AddVar(varName : string, defaultValue : any)

service.GetVarValue(varName : string, defaultValue : any)

service.GetVarSignals(varName : string)

service.SetVar(varName : string, value : any)

Signals:

var.OnChange(func : function)

var.OnIncrease(func : function)

var.OnDecrease(func : function)

Example usage:

Script1:


local Stic = require(game.ReplicatedStorage.Stic)
local someService = Stic.NewService('someService')
function addFunc(n1,n2)
    return n1+n2
end
someService.AddFunction('add', addFunc)
local someVar = someService.AddVar('someVar') -- if you don't give second parameter, it means nil
wait(2)
someVar.SetVar('someVar', 10)

Script2:


local Stic = require(game.ReplicatedStorage.Stic)
wait() -- we wait so the service is created first
local someService = Stic.GetService('someService')
print(someService.add(1+2)) -- prints 3
local signalsOfSomeVar = Stic.GetVarSignals('someVar')
function someVarChanged(newValue)
    print(newValue) -- prints 10 (the new value)
end
signalsOfSomeVar.OnChange(someVarChanged)
--OnIncrease and OnDecrease works the same way

Services are the second most useful things in Stic after signals. They are very cool!
